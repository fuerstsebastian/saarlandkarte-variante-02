# Saarland Explorer

Interaktive Karte archäologischer Fundstellen, Bodendenkmäler, Museen und historischer Orte im Saarland.

Diese Anwendung ist ein studentisches Projekt, das im Rahmen der Lehrveranstaltung "Karten, Quellen, KI: Digitale Erschließung archäologischer Attraktionen im Saarland" im Sommersemester 2026 am Lehrstuhl für Vor- und Frühgeschichte der Universität des Saarlandes entstanden ist.

## Lokale Entwicklung

**Voraussetzungen:** Node.js

1. Abhängigkeiten installieren:
   `npm install`
2. Entwicklungsserver starten:
   `npm run dev`

## Prototyp-Status

Diese Anwendung befindet sich im Prototyp-Status und ist eine Übungs- und Lehrveranstaltungsarbeit. Die dargestellten Fundstellendaten und Beschreibungen werden laufend ergänzt und korrigiert.
