2. Hosting (GitHub Pages)

Diese Webanwendung wird als statisches Projekt über GitHub Pages bereitgestellt, einem Dienst der GitHub Inc. (88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA). Beim Aufruf der Seite baut Ihr Browser eine Verbindung zu den Servern von GitHub auf. Dabei kann GitHub die IP-Adresse der Besuchenden zu Sicherheitszwecken verarbeiten und loggen. In den Server-Logfiles werden typischerweise erfasst:

  • IP-Adresse des zugreifenden Endgeräts
  • Datum und Uhrzeit des Abrufs
  • Name und URL der abgerufenen Datei
  • Übertragene Datenmenge
  • Referrer-URL (die zuvor besuchte Seite)
  • Verwendeter Browser und Betriebssystem

Diese Verarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO zur Gewährleistung der Betriebssicherheit und Stabilität des Webservers. Weitere Informationen finden Sie in den GitHub Privacy Statements.

3. Kartenmaterial (OpenStreetMap / Pigeon Maps)

Diese Anwendung bindet interaktives Kartenmaterial von OpenStreetMap (OSM) ein, um archäologische Fundstellen geografisch darzustellen. OpenStreetMap wird von der OpenStreetMap Foundation (OSMF), St John's Innovation Centre, Cowley Road, Cambridge, CB4 0WS, Großbritannien, bereitgestellt. Beim Laden der Karte fordert Ihr Browser Kartendaten (Kacheln) direkt von den Servern der OSMF an. Dabei wird Ihre IP-Adresse und ggf. weitere technische Browserdaten an die Server der OSMF übertragen. Es werden dabei keine Cookies oder persistenten Tracker durch OSM gesetzt. Weitere Informationen finden Sie in der Datenschutzerklärung der OpenStreetMap Foundation.

4. Weitere externe Dienste

a) Routenberechnung (Open Source Routing Machine / OSRM)

Für die Berechnung von benutzerdefinierten Wegbeschreibungen und Wandertouren nutzt diese Anwendung die Open Source Routing Machine (OSRM). Bei jeder Routenberechnung werden die Start- und Zielkoordinaten an die öffentliche OSRM-API unter router.project-osrm.org übertragen. Um die Berechnung durchzuführen und das Ergebnis an Ihren Browser zurückzusenden, verarbeitet OSRM Ihre IP-Adresse sowie die geographischen Wegpunkte. Diese Abfrage ist für die Erbringung der vom Nutzer ausdrücklich angeforderten Routenplanungs-Funktion erforderlich (Art. 6 Abs. 1 lit. b und lit. f DSGVO).

b) Externe Schriftarten (Google Fonts)

Diese Anwendung bindet externe Schriften von Google Fonts ein, um ein einheitliches und ästhetisches Erscheinungsbild zu gewährleisten. Beim Laden der Seite lädt Ihr Browser die benötigten Webschriften direkt von den Google-Servern herunter, wobei Google Kenntnis von Ihrer IP-Adresse erlangt. Dies erfolgt im Interesse einer optisch einwandfreien Darstellung unserer Seiten (Art. 6 Abs. 1 lit. f DSGVO).

c) Bilddaten (Unsplash)

Einige repräsentative Bilder der Epochen und Fundstellen werden direkt über das Content Delivery Network von Unsplash (Unsplash Inc., Montreal, Kanada) geladen. Hierbei baut Ihr Browser eine Verbindung zu Unsplash-Servern auf, wodurch Ihre IP-Adresse dorthin übermittelt wird.

5. Lokaler Browserspeicher (LocalStorage)

Zur Speicherung Ihrer bevorzugten Einstellungen (z. B. gewählter Farbmodus, Filtereinstellungen) nutzt diese Anwendung den lokalen Speicher Ihres Browsers (LocalStorage). Diese Daten verbleiben ausschließlich auf Ihrem Gerät, werden nicht an Server übertragen und dienen allein dem Nutzungskomfort (Art. 6 Abs. 1 lit. f DSGVO).

6. Keine Tracking-Cookies und Analysetools

Diese Anwendung verzichtet vollständig auf den Einsatz von Tracking-Cookies, Werbe-Pixeln oder Webanalyse-Diensten (wie z. B. Google Analytics). Es werden keine Nutzerprofile erstellt. Es findet kein Nutzer-Tracking statt.
